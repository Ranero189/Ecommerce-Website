# Ecommerce-Website
Construir e implantar um site de Ecommerce (Eco-Fashion) com HTML, CSS e JavaScript.

## Propósito Ambiental
Este projeto atua como uma solução proativa contra os impactos negativos do fast-fashion. O e-commerce foca na venda de roupas feitas com materiais reciclados, algodão orgânico e processos de baixo impacto, além de conscientizar o usuário sobre os impactos poluentes da indústria têxtil tradicional.
